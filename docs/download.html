Installation on Windows

1. Download Geniemoji For Windows from above

2. Now, you will see a file called Geniemoji Setup, double click that file

3. Now, you may see something like Windows Protected Your PC

4. Click on More Info and Run Anyway

5. Next, there may be a popup window with this text: Do you want to allow this app to make changes...

6. Simply click Yes or Allow, whichever one you see.

7. Finally, follow the steps of the setup and you are good to use the app!

Installation on MacOS

1. Download Geniemoji For MacOS from above

2. Once downloaded, double-click the file

3. This will extract the app. Simply drag the app into the Applications folder on your machine.

4. If you are prompted with something like Geniemoji cannot be opened because the developer cannot be verified., click cancel.

5. Go to system preferences -> Security and Privacy -> General -> and click Open Anyway.
